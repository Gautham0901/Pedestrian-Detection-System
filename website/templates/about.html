About the Project

System Architecture

Detection Layer

Utilizes YOLOv10 for accurate real-time pedestrian detection with enhanced performance through:

  • Advanced feature extraction
  • Multi-scale detection capabilities
  • Optimized inference pipeline

Tracking Integration

Implements DeepSORT (Deep Simple Online Realtime Tracking) algorithm with:

  • Integrated detection and tracking pipeline
  • Efficient ID management system
  • Robust occlusion handling

XAI Integration

Explainable AI component provides transparency through:

  • Real-time saliency map generation
  • Feature importance visualization
  • Decision confidence metrics

Technology Stack

Our system utilizes the following technologies:

  • Object Detection: YOLOv10 (our primary model) with support for YOLOv8
  • Tracking: DeepSORT algorithm for reliable multi-object tracking
  • Backend: Python with Flask for web service integration
  • Frontend: HTML5, CSS3 and JavaScript with Bootstrap framework
  • Data Analysis: NumPy, Pandas, and custom analytics modules
